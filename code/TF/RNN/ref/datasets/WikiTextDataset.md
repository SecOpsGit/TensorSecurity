```
The WikiText Long Term Dependency Language Modeling Dataset

WikiText 英語詞庫資料（The WikiText Long Term Dependency Language Modeling Dataset）
是一個包含1億個詞彙的英文詞庫資料，這些詞彙是從Wikipedia的優質文章和標杆文章中提取得到，
包括WikiText-2和WikiText-103兩個版本，
相比於著名的 Penn Treebank (PTB) 詞庫中的詞彙數量，前者是其2倍，後者是其110倍。

每個詞彙還同時保留產生該詞彙的原始文章，這尤其適合當需要長時依賴(long term dependency)自然語言建模的場景。
```

# Google BERT
```
Google公司AI團隊新發佈的BERT模型，在機器閱讀理解頂級水準測試SQuAD1.1中表現出驚人的成績：
全部兩個衡量指標上全面超越人類
在11種不同NLP測試中創出最佳成績，包括將GLUE基準推至80.4％（絕對改進7.6％），MultiNLI準確度達到86.7% （絕對改進率5.6％）等。
BERT將為NLP帶來里程碑式的改變，也是NLP領域近期最重要的進展。
```
